Type: itemCombinationLoadouts
Example file: item_combination_loadout.example.player.json

What this data does:
It groups recipe ids into a loadout that can be assigned to player stats or another inventory profile.

Fields shown in the example:

- `combinationLoadoutId`: canonical loadout id.
- `combinationIds`: recipe ids available through this loadout. The example enables `item_combination.example.training_arrow_bundle`.
