Type: itemBoxSpawnLoadouts
Example file: item_box_spawn_loadout.example.stage_01.json

What this data does:
It groups item box spawn ids into a loadout that a stage can request.

Fields shown in the example:

- `id`: canonical item-box spawn loadout id.
- `itemBoxSpawnIds`: item box spawn ids included in this loadout. The example includes `item_box_spawn.example.stage_01.001`.
