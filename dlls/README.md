# DLL References

These DLLs are reference assemblies for experimental Code/DLL mods.

Use DLLs from the same game version you target.

Included:

1. NurtaleNesche.Modding.Abstractions.dll
2. NurtaleNesche.Modding.Validation.dll
3. NurtaleNesche.Runtime.dll
4. UnityEngine.CoreModule.dll
5. Newtonsoft.Json.dll

Do not upload Assembly-CSharp.dll as the public SDK.

If a modder builds against DLLs from a different game version, Code/DLL mods may compile but fail at runtime.
