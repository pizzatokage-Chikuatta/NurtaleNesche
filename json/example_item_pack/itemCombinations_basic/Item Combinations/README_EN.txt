Type: itemCombinations
Example file: item_combination.example.training_arrow_bundle.json

What this data does:
It defines one crafting/combination recipe.

Fields shown in the example:

- `combinationId`: canonical recipe id.
- `inputs`: item ids consumed or checked by the recipe. The example requires two `item.example.training_arrow` entries.
- `output`: item id created by the recipe. The example creates `item.wooden_arrow`.
