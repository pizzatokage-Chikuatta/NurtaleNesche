Type: interactionOptionMeta
Example file: example_training_crate_options.json

What this data does:
It defines visible interaction menu options.

Fields shown in the example:

- `id`: interaction command id used by an interaction loadout.
- `en`: English text shown for the option.
- `order`: display order. Lower numbers appear earlier than higher numbers.

The example adds `Open crate` and `Leave it` options for a training crate profile.
