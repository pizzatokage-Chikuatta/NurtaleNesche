Type: audioFiles
Example file: sfx.clip.example_ui_confirm.json

What this data does:
It registers one SFX clip id and the audio file path used to load it.

Fields shown in the example:

- `id`: canonical SFX clip id. Classification records refer to this value through `clipId`.
- `file`: audio file path. The example points to `Audio/Sound Effects/Audio Files/Example/example_ui_confirm.ogg`.
