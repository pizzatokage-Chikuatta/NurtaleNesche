Type: bgmStageLoadouts
Example file: bgm_stage_loadout.example.stage_01.json

What this data does:
It assigns a BGM track id to a stage id.

Fields shown in the example:

- `stageId`: stage that receives the BGM assignment. The example uses `stage_01`.
- `bgmTrackId`: BGM track id to play for that stage. The example uses `bgm.example.training_room`.
