アイテム名
================================
アイテム名の表示テキストを変更するModを作ります。

この対象は「Use_With_Caution」です。便利ですが壊しやすい項目もあるため、まずは既存データの上書きから始めるのを推奨します。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_Chikuatta\Lootings\Mod Item Names\Item Names

①"Mod_Chikuatta"は例です。個人のニックネームを使用してください。

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Item Names'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。

```json
{
  "experimentalApiVersion": "v1",
  "type": "itemNames",
  "entry": "Item Names",
  "overwriteExisting": true
}
```

①今回はアイテム名のModなので"type"は"itemNames"である。
②"entry"は横のフォルダー名と一致しないといけない。今回なら"Item Names"である。

ステップ4
--------
以下のフォルダーから'item.biscuit.json'をコピーし、
StreamingAssets\Lootings\Items And Equipments\Item Names
以下へ張り付ける。
Mods\Mod_Chikuatta\Lootings\Mod Item Names\Item Names

①'item.biscuit.json'は例として選んでおり、同じ種類の他ファイルを選んでも手順は同じです。

ステップ5
--------
'item.biscuit.json'の中身を修正する。
①"itemId": 対象アイテム/装備id。変えたい対象と一致させる。
②"unknownNames": 未鑑定/未知状態などで表示する名前。言語キーごとに文字列を書く。
③"knownNames": 鑑定済み/既知状態などで表示する名前。空の場合はunknownNames側が使われることがある。
④"jp"/"en"/"cn"/"tw"/"kr"/"de"/"ru": 各言語キー。必要な言語だけ変更してよい。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①名前を変えるだけなら比較的安全。itemIdを間違えると対象に反映されない。
②新規アイテムを追加する場合は、Item DefinitionsとItem Namesの両方を用意する。
