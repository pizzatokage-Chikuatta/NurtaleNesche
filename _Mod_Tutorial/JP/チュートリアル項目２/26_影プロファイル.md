影プロファイル
================================
キャラクターやオブジェクトの影の見た目を変更するModを作ります。

この対象は「Use_With_Caution」です。便利ですが壊しやすい項目もあるため、まずは既存データの上書きから始めるのを推奨します。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_Chikuatta\Patrollers\Mod Shadow Profiles\Shadow Profiles

①"Mod_Chikuatta"は例です。個人のニックネームを使用してください。

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Shadow Profiles'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。

```json
{
  "experimentalApiVersion": "v1",
  "type": "shadowProfiles",
  "entry": "Shadow Profiles",
  "overwriteExisting": true
}
```

①今回は影プロファイルのModなので"type"は"shadowProfiles"である。
②"entry"は横のフォルダー名と一致しないといけない。今回なら"Shadow Profiles"である。

ステップ4
--------
以下のフォルダーから'Goblin_ShadowPropertySO.json'をコピーし、
StreamingAssets\Shadow\Shadow Property Json
以下へ張り付ける。
Mods\Mod_Chikuatta\Patrollers\Mod Shadow Profiles\Shadow Profiles

①'Goblin_ShadowPropertySO.json'は例として選んでおり、同じ種類の他ファイルを選んでも手順は同じです。

ステップ5
--------
'Goblin_ShadowPropertySO.json'の中身を修正する。
①"id": shadow profile id。Patroller Dataなどから参照される。
②"default": どのstateにも一致しない時の基本影設定。
③"scale": 影の横/縦サイズ。
④"offset": 影の位置ズレ。
⑤"alpha": 影の透明度。0で見えない、1に近いほど濃い。
⑥"smooth": 影設定の切り替わりを滑らかにする値。
⑦"states": Animator state名ごとの影設定。state名が間違っていると反映されない。
⑧"isBlendTree"/"clips": BlendTree state用の細かい設定。上級者向け。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①影は見た目だけなので比較的安全だが、state名を間違えると狙ったstateに反映されない。
②最初はdefaultのscale/offset/alphaだけ調整するのを推奨。
