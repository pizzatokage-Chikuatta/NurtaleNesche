ポーション効果
================================
ポーションを飲んだ時/当たった時の効果を変更するModを作ります。

この対象は「Use_With_Caution」です。便利ですが壊しやすい項目もあるため、まずは既存データの上書きから始めるのを推奨します。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_Chikuatta\Lootings\Mod Potion Effects\Potion Effects

①"Mod_Chikuatta"は例です。個人のニックネームを使用してください。

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Potion Effects'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。

```json
{
  "experimentalApiVersion": "v1",
  "type": "potionEffects",
  "entry": "Potion Effects",
  "overwriteExisting": true
}
```

①今回はポーション効果のModなので"type"は"potionEffects"である。
②"entry"は横のフォルダー名と一致しないといけない。今回なら"Potion Effects"である。

ステップ4
--------
以下のフォルダーから'item.potion.aphrodisia.json'をコピーし、
StreamingAssets\Lootings\Items And Equipments\Potions
以下へ張り付ける。
Mods\Mod_Chikuatta\Lootings\Mod Potion Effects\Potion Effects

①'item.potion.aphrodisia.json'は例として選んでおり、同じ種類の他ファイルを選んでも手順は同じです。

ステップ5
--------
'item.potion.aphrodisia.json'の中身を修正する。
①"potionId": 対象ポーションのitem id。Item Definitions側のidと一致させる。
②"onDrink": 飲んだ時の効果。consumeItem/actions/vfxIdなどを持つ。
③"consumeItem": trueなら使用時にアイテムを消費する。
④"actions": 実行する効果リスト。例: AddStatusでstatus effectを付与する。
⑤"effectId": 付与するStatus Effect ID。Reference_Dataフォルダー内の「NurtaleNesche_Modding_Reference_IDs.md」を参照。
⑥"onHit": 投げたポーションが当たった時の効果。stunTime/actions/vfxIdなどを持つ。
⑦"vfxId": 再生するVFX id。既存idを使うのが安全。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①存在しないStatus Effect IDを入れると効果が出ない。
②新しいポーションを作る場合はItem Definitions、Item Names、Potion Effects、ドロップテーブルなども必要になる。
③アクションtypeはコード側で対応しているものだけ使える。適当に新しいtypeを書いても動かない。
