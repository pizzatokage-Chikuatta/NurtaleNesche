ステータスメタデータ
================================
ステータス効果のメタデータを変更するModを作ります。拘束、呪い、視界妨害などの効果設定を変えるためのデータです。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Status\Mod Status Metadata\Status Metadata

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Status Metadata'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "statusMetadata",
  "entry": "Status Metadata",
  "overwriteExisting": true
}
```
①今回はステータスメタデータのModなので"type"は"statusMetadata"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
以下のフォルダーから'status_effect.eyemask.json'などをコピーし、
StreamingAssets\Status\Status Metadata
以下へ張り付ける。
Mods\Mod_Chikuatta\Status\Mod Status Metadata\Status Metadata

ステップ5
--------
コピーしたステータスメタデータJsonの中身を修正する。
①"id": 変えてはいけない。既存ステータスを書き換えるためのIdである。
②"uiIconSpriteId": UIに出すアイコンSprite Id。
③"type": ステータスの種類。例: "physical"など。
④"isCureable": trueの場合、治療対象になる。
⑤"isIncludeInCursePool": trueの場合、呪い候補に含まれる。
⑥"stageAvailableFrom": そのステータスが出始めるステージ番号。
⑦"lockLevel": 解除難易度に使う値。
⑧"efficacy": 実際に発生する効果Idのリスト。例: "efficacy.blind"など。
⑨"blockedFrom": このステータスが有効な時にブロックするステータスIdのリスト。
⑩"forceDeactivates": このステータスが有効になった時に強制解除するステータスIdのリスト。
⑪"escapeTools": 解除に使える道具Idのリスト。例: "escape_tool.lockpick"。
⑫"animationOwned": trueの場合、このステータスが表示トラックを所有する。
⑬"bodyProfileId": 体表示Profile Id。基本は既存値のまま。
⑭"bodyProfileHardOverride": trueの場合、体表示Profileを強制的に上書きする。
⑮"visibleTracks": 表示するアニメーションTrack Idのリスト。例: "eyemask"。
⑯"stackTrackRules": スタック数に応じて表示Trackを変える上級者向け設定。
⑰"overrides": 対象Profileごとの上書き設定。プレイヤー用、友好捕虜用などで表示や効果を変える。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①"efficacy"を変えるとゲーム挙動が大きく変わる。まずは既存Jsonをコピーして一箇所だけ変えるのが安全。
②"overrides"内にも"efficacy"や"visibleTracks"がある場合、対象Profileではそちらが使われることがある。
③表示だけ変えたい場合は"visibleTracks"、動作を変えたい場合は"efficacy"を見る。
