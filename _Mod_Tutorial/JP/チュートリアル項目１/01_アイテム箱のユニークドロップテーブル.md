アイテム箱のユニークドロップテーブル
================================
アイテム箱の「ユニークドロップテーブル」のModを作ります。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Lootings\Mod Drop Tables\Drop Tables\Stage 1\Item Box

①"Mod_Chikuatta"は例です。個人のニックネームを使用してください。
②"Stage 1\Item Box"はステージ1のアイテム箱用ドロップテーブルを置く場所です。他ステージを変えたい場合は対応するステージフォルダーを使う。

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Drop Tables'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて（Visual Studio Code推奨）中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "dropTables",
  "entry": "Drop Tables",
  "overwriteExisting": true
}
```
①今回はドロップテーブルのModなので"type"は"dropTables"である。
②"entry"は横のフォルダー名と一致しないといけない。今回なら"Drop Tables"である。
③"overwriteExisting": true にすることで、同じ"id"の built-in ドロップテーブルをMod側で上書きする。

ステップ4
--------
以下のフォルダーから'drop_table.item_box.stage_01.0.json'をコピーし、
StreamingAssets\Lootings\Drop Tables\Stage 1\Item Box
以下へ張り付ける。
Mods\Mod_Chikuatta\Lootings\Mod Drop Tables\Drop Tables\Stage 1\Item Box

①'drop_table.item_box.stage_01.0.json'は例として選んでおり、同じフォルダー内の他のドロップテーブルを選んでも手順は同じです。

ステップ5
--------
'drop_table.item_box.stage_01.0.json'の中身を修正する。
初期状態は以下のような形である。
```json
{
  "id": "drop_table.item_box.stage_01.0",
  "guaranteedItem": [ "item.biscuit", "item.lockpick" ],
  "guaranteedEquipment": [],
  "groups": [
    {
      "rolls": 2,
      "entries": [
        { "itemId": null, "weight": 1 },
        { "itemId": "item.biscuit", "weight": 1 },
        { "itemId": "item.lockpick", "weight": 1 }
      ]
    }
  ]
}
```

①"id": 変えてはいけない。StreamingAssets内でこれと一致するidのドロップテーブルが書き換えられる。
②"guaranteedItem": 必ずドロップするアイテムidのリスト。
③"guaranteedEquipment": 必ずドロップする装備idのリスト。
④この例では、ビスケット1つとロックピック1つが必ず出る。
⑤"groups": ランダムドロップの設定リスト。
⑥"rolls": サイコロを振る回数。2ならこのグループから2回抽選する。
⑦"entries": 抽選候補リスト。
⑧"itemId": 出るアイテムid。"itemId": null は何も出ない（外れ）。
⑨"weight": 選ばれやすさ。大きいほど選ばれやすい。
⑩アイテム/装備のidはReference_Dataフォルダー内の「NurtaleNesche_Modding_Reference_IDs.md」を参照。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①なぜ「アイテム箱のドロップテーブル」ではなく「アイテム箱のユニークドロップテーブル」なのかというと、アイテム箱は主に「ステージ共通ドロップテーブル」と「アイテム箱ユニークドロップテーブル」の両方からアイテム/装備を落とすためである。
②つまり、今回のユニークドロップテーブルを空にしても、ステージ共通ドロップから何か出る可能性がある。
③特定のアイテム箱だけ中身を変えたい場合は、このユニークドロップテーブルを編集する。
④ステージ全体の共通ドロップを変えたい場合は、StreamingAssets\Lootings\Drop Tables の各ステージフォルダーや Common フォルダー内のドロップテーブルを編集する。
⑤ステージ共通ドロップは複数のドロップテーブルをまとめて使うため、ドロップテーブルロードアウトも関係する。
