効果音ファイル
================================
新しい効果音ファイル（.ogg または .wav）を読み込ませるModを作ります。
この手順だけでは音は鳴りません。次の「効果音分類ID」でその効果音を使う必要があります。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Audio\Sound Effects\Mod SFX\Audio Files\Nesche\Footsteps

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod SFX'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "audioFiles",
  "entry": "Audio Files",
  "overwriteExisting": true
}
```
①今回は効果音ファイルのModなので"type"は"audioFiles"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
使いたい.oggまたは.wavファイルを以下へ入れる。
Mods\Mod_Chikuatta\Audio\Sound Effects\Mod SFX\Audio Files\Nesche\Footsteps
例: My_Run_1.ogg

ステップ5
--------
同じフォルダーに'sfx.clip.nesche_run_1.json'などのJsonを作る。
```json
{
  "id": "sfx.clip.nesche_run_1",
  "file": "My_Run_1.ogg"
}
```
①"id": 効果音クリップのId。既存Idと同じにすると、その効果音クリップを置き換える。
②"file": 実際の.ogg/.wavファイルへのパス。同じフォルダーに置くならファイル名だけでよい。
③ファイル名が'sfx.clip.'から始まるJsonの場合、idが省略されてもファイル名がIdとして使われるが、初心者は必ず"id"を書く方が安全。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①"audioFiles"は「音素材を登録する」だけである。
②実際にどの場面で鳴るかは"classificationID"側で決まる。
③既存の効果音を差し替えたいだけなら、既存のsfx.clip.* Idと同じIdにする。

ファイル参照の流れ
--------

ファイルを登録することと、ゲーム内で使わせることは別です。例えば音声は、audio file -> clip ID -> classification/track/loadout -> gameplay reference のように参照がつながります。
