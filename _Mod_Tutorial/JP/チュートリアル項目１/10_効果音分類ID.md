効果音分類ID
================================
効果音分類IDのModを作ります。これは「この場面で、どの効果音クリップ候補を鳴らすか」を決めるデータです。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Audio\Sound Effects\Mod Classification ID\Classification ID\Player

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Classification ID'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "classificationID",
  "entry": "Classification ID",
  "overwriteExisting": true
}
```
①今回は効果音分類IDのModなので"type"は"classificationID"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
以下のフォルダーから'sfx.nesche_run.json'などをコピーし、
StreamingAssets\Audio\Sound Effects\Classification ID\Player
以下へ張り付ける。
Mods\Mod_Chikuatta\Audio\Sound Effects\Mod Classification ID\Classification ID\Player

ステップ5
--------
'sfx.nesche_run.json'の中身を修正する。
```json
{
  "id": "sfx.nesche_run",
  "clips": [
    { "clipId": "sfx.clip.nesche_run_1", "weight": 1 }
  ]
}
```
①"id": 変えてはいけない。既存の分類IDを書き換えるためのIdである。
②"clips": この分類IDで鳴らす効果音クリップ候補のリスト。
③"clipId": 効果音ファイル側で登録したsfx.clip.* Id。
④"weight": 大きいほどその効果音が選ばれやすい。全部1なら同じ確率。
⑤候補を一つだけにすると、必ずその効果音だけが鳴る。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①新しい効果音を使う場合は、先に"audioFiles"でsfx.clip.*を登録する必要がある。
②既存の分類IDをコピーして"clips"だけ変えるのが一番安全。

ファイル参照の流れ
--------

ファイルを登録することと、ゲーム内で使わせることは別です。例えば音声は、audio file -> clip ID -> classification/track/loadout -> gameplay reference のように参照がつながります。
