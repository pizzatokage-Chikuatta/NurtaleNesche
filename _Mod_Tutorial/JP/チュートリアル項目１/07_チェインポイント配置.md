チェインポイント配置
================================
チェインポイントをステージ開始時に配置するModを作ります。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Chainpoints\Mod Initial Spawns\Initial Spawns

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Initial Spawns'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "chainPointSpawns",
  "entry": "Initial Spawns",
  "overwriteExisting": true
}
```
①今回はチェインポイント配置のModなので"type"は"chainPointSpawns"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
既存のチェインポイント配置Jsonがある場合はそれをコピーする。新規追加する場合は以下のようなJsonを作る。
```json
{
  "id": "chainpoint_spawn.kabe_manguri.stage_03.001",
  "checkpointUuid": "ここに新規GUIDを書く",
  "enabled": true,
  "regionId": "5e944ee6-e478-4e41-ba0c-ece73dd377f9",
  "offsetX": 0,
  "heightFromGround": 2.8,
  "rotationY": 0,
  "chainPointId": "chainpoint.kabe_manguri"
}
```

ステップ5
--------
チェインポイント配置Jsonの中身を修正する。
①"id": 配置データのId。新規追加なら他と被らないIdにする。
②"checkpointUuid": この配置専用のGUID。新規追加なら必ず新しいGUIDにする。
③"enabled": falseの場合このチェインポイントはスポーンしない。
④"regionId": 配置する地域Id。Reference_Dataフォルダー内の「NurtaleNesche_Modding_Region_UUIDs.md」を参照。
⑤"offsetX": 地域の中心から左右にどれだけズラして配置するか。プラスで右、マイナスで左。
⑥"heightFromGround": 地面からの高さ。チェインポイントが床に埋まる/浮く時に調整。
⑦"rotationY": Y回転。向きを反転したい時などに使う。
⑧"chainPointId": どのチェインポイント本体を使うかのId。例: "chainpoint.kabe_manguri"。
⑨"statusId": 必要なチェインポイントだけが使う上級者向け設定。基本は書かない。
⑩"typeEnum": 必要なチェインポイントだけが使う上級者向け設定。基本は書かない。
⑪"isNeedLockPickToEscape": 必要なチェインポイントだけが使う上級者向け設定。基本は書かない。

GUIDの作り方
--------
Windows PowerShellを開いて以下を実行する。
```powershell
[guid]::NewGuid().ToString()
```
表示された文字列を"checkpointUuid"へ入れる。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①チェインポイント配置Jsonを作っただけではステージに出ない場合がある。その時はチェインポイント配置ロードアウトにもIdを追加する。
②使用できる"chainPointId"はStreamingAssets\Chainpoints\Definitions内のJsonで確認できる。
③例: "chainpoint.cuff_chain_on_wall", "chainpoint.cuff_chain_with_lever_on_wall", "chainpoint.guillotine", "chainpoint.kabeshiri", "chainpoint.kabeshiri_front", "chainpoint.kabe_manguri", "chainpoint.milk_machine", "chainpoint.mouthpiece_chain_on_wall"。
