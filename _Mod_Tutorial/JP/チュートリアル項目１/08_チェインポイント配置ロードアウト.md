チェインポイント配置ロードアウト
================================
チェインポイント配置ロードアウトのModを作ります。これはステージ開始時に、どのチェインポイント配置Idを使うかを決めるリストです。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Chainpoints\Mod Initial Spawn Loadouts\Initial Spawn Loadouts

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Initial Spawn Loadouts'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "chainPointSpawnLoadouts",
  "entry": "Initial Spawn Loadouts",
  "overwriteExisting": true
}
```
①今回はチェインポイント配置ロードアウトのModなので"type"は"chainPointSpawnLoadouts"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
以下のフォルダーから'chainpoint_spawn_loadout.stage_03.json'などをコピーし、
StreamingAssets\Chainpoints\Initial Spawn Loadouts
以下へ張り付ける。
Mods\Mod_Chikuatta\Chainpoints\Mod Initial Spawn Loadouts\Initial Spawn Loadouts

ステップ5
--------
'chainpoint_spawn_loadout.stage_03.json'の中身を修正する。
①"chainPointSpawnLoadoutId": 変えてはいけない。対象ステージのロードアウトを書き換えるためのIdである。
②"chainPointSpawnIds": ステージ開始時に配置するチェインポイント配置Idのリスト。
③新しく作ったチェインポイント配置をステージに出したい場合、その配置Jsonの"id"をここへ追加する。
④ここにIdを書かないと、チェインポイント配置Jsonが存在していてもステージには出ない。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①ステージによっては初期チェインポイントが空のロードアウトになっている場合がある。その場合は"chainPointSpawnIds"に自分の配置Idを追加する。
②配置できない特殊チェインポイントもあるため、まずはDefinitions内に存在する通常チェインポイントから試すのが安全。

置き換えリストの注意
--------

このファイルは「少し追加するパッチ」ではなく、リスト全体の置き換えです。同じ"id"を上書きする場合、最終的に使われるリストはMod側に書いたリストになります。残したいbuilt-in項目は消さずに残してください。
