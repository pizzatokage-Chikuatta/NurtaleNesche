プレイヤーステータス
================================
プレイヤーの基本ステータスを変更するModを作ります。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_｛個人のニックネーム｝\Player\Mod Player Stats\Player Stats

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Player Stats'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "playerStats",
  "entry": "Player Stats",
  "overwriteExisting": true
}
```
①今回はプレイヤーステータスのModなので"type"は"playerStats"である。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
以下のフォルダーから'player_stats.nesche_default.json'をコピーし、
StreamingAssets\Player\Stats
以下へ張り付ける。
Mods\Mod_Chikuatta\Player\Mod Player Stats\Player Stats

ステップ5
--------
'player_stats.nesche_default.json'の中身を修正する。
①"id": 変えてはいけない。通常は"player_stats.nesche_default"を書き換える。
②"displayName": 表示名。基本変えなくてよい。
③"actionRuleProfileId": 行動ルールProfile Id。基本変えない。
④"shadowProfileId": 影の表示Profile Id。基本変えない。
⑤"vitals.maxHealth": 最大HP。
⑥"vitals.healthCap": HP上限。maxHealthより小さくしない。
⑦"vitals.maxStamina": 最大スタミナ。
⑧"vitals.staminaCap": スタミナ上限。maxStaminaより小さくしない。
⑨"vitals.staminaRestoreAmount": スタミナ回復量。
⑩"movement.walkSpeed": 歩き速度。
⑪"movement.sprintMultiplier": 走り倍率。
⑫"interaction.interactIntervalSeconds": インタラクト連打間隔。小さすぎる値は非推奨。
⑬"inventory.maxItemSpeciesHoldable": 持てるアイテム種類数。
⑭"inventory.combinationLoadoutId": アイテム合成ロードアウトId。基本変えない。
⑮"sexualHeat.decreasePerSecond": 性的興奮度の自然減少量。
⑯"extras": 追加用。基本空のままでよい。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

備考
--------
①ゲームバランスに直結するため、数値を極端にしすぎるとイベントや敵挙動のテストが難しくなる。
②まずはHP、スタミナ、歩き速度だけを小さく変更して動作確認するのが安全。
