インタラクションオプションメタデータ
================================
インタラクション選択肢の表示テキストや表示順のModを作ります。これは表示定義であり、実際の処理を新規追加するものではありません。

ステップ1
--------
'Mods'の中に以下のフォルダー階層を作る。
Mods\Mod_Chikuatta\Advanced\Mod Interaction Option Metadata\Interaction Option Metadata

ステップ2
--------
Mods\mod.json templatesから'mod.template_folder_backed.json'をコピーし、'Mod Interaction Option Metadata'の中に張り付け、名前を'mod.json'へ変更する。

ステップ3
--------
mod.jsonを開いて中身を以下のように変更する。
```json
{
  "experimentalApiVersion": "v1",
  "type": "interactionOptionMeta",
  "entry": "Interaction Option Metadata",
  "overwriteExisting": true
}
```
①今回はインタラクションオプションメタデータのModなので"type"はこの値にする。
②"entry"は横のフォルダー名と一致しないといけない。

ステップ4
--------
以下のフォルダーから'item_interaction_options.json'をコピーし、
StreamingAssets\Interaction\Option Metadata
以下へ張り付ける。
Mods\Mod_Chikuatta\Advanced\Mod Interaction Option Metadata\Interaction Option Metadata

ステップ5
--------
'item_interaction_options.json'の中身を修正する。
1. "id": インタラクション選択肢のId。既存の表示を書き換える場合は変えてはいけない。
2. "jp"/"en"など: 各言語で表示されるテキスト。
3. "order": 表示順。小さいほど上に出やすい。
4. "icon"などがある場合: 表示用情報。処理本体ではない。

最後にゲームのメインメニュー -> Mods -> Mod Reportで正しく認識されているか確認することで完了。

注意
--------
1. ここを編集しても、新しい行動処理そのものは追加されない。
2. idがInteraction Loadout側のcommand/optionと一致していないと表示されない。
3. 表示だけ変えたい時に使う。挙動を変えたい場合は別のデータも必要。

コピーしてから少しずつ変えるルール
--------

この上級者向けデータは、まずbuilt-inファイルをコピーして一項目ずつ変更してください。空のJSONから作ると、runtime codeや他registryへの隠れた参照を見落としやすいです。
