言語ファイル
================================

このチュートリアルでは新しい言語項目とCSVを追加します。言語Modは普通のfolder-backed JSON modとは少し違い、言語メタデータはmod.jsonへ直接書きます。

ステップ1
--------

Modsの中に以下のフォルダーを作る。

~~~text
Mods\Mod_YourName\Language\Pirate English
~~~

ステップ2
--------

Pirate Englishフォルダーの中にmod.jsonを作る。

~~~json
{
  "experimentalApiVersion": "v1",
  "type": "language",
  "name": "Pirate English",
  "language": "Pirate English",
  "languageCode": "PIR"
}
~~~

項目の意味：

1. "type": "language" で言語を登録する。
2. "name": Mod名。
3. "language": 言語ドロップダウンに表示される名前。
4. "languageCode": CSVファイル名に使うコード。

ステップ3
--------

チャット/システム文を変える場合、mod.jsonの横に以下を作る。

~~~text
CSV_ChatMessage_PIR.csv
~~~

例：

~~~csv
ItemGet,{0} plundered {1}.,,
ItemUse,{0} uses {1}.,,
Normal,The PizzaTokage is staring at you.,,
~~~

1列目がkey、2列目が表示テキストです。

重要：CSVの形は種類ごとに違います。ChatMessage CSVとSpeech CSVは列の意味が違います。別種類のCSVの行をそのまま流用しないでください。

ステップ4
--------

ステージ名などmisc CSVを変える場合、以下のフォルダーを作る。

~~~text
MISC CSV
~~~

ステージ名なら以下を作る。

~~~text
CSV_Stage Name_PIR.csv
~~~

languageCodeがPIRなら、ファイル名もPIRにする必要があります。ENなど別コードにしないでください。

ステップ5
--------

パトローラーのセリフを変える場合、MISC CSVの直下に以下のようなファイルを作る。

~~~text
CSV_Speech_PIR_patroller.goblin.csv
~~~

例：

~~~csv
patroller.goblin,Goblin,patrol_001,Patrol,1,Arrr...
patroller.goblin,Goblin,suspicion_001,Suspicion,1,Who goes there?!
~~~

Speech CSVの列：

1. speakerId
2. speakerDisplayName
3. speechId
4. action
5. priority
6. speech

重要CSVルール
--------

1. UTF-8で保存する。
2. 最初は単純な行でテストする。
3. テキスト内のカンマに注意する。CSV分割が壊れる場合がある。
4. languageCodeとファイル名のsuffixを一致させる。
5. Speech CSVはMISC CSV\SpeechではなくMISC CSV直下へ置く。

よくあるミス
--------

1. language.jsonを作る。v1ではmod.jsonから読む。
2. languageCodeはPIRなのにCSV_ChatMessage_EN.csvを作る。
3. Speech CSVをMISC CSV\Speechへ入れる。
4. UTF-8ではない文字コードで保存する。
