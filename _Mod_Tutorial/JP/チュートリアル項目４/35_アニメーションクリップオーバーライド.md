アニメーションクリップオーバーライド
================================

このチュートリアルでは、既存のcontroller slotを別のclipIdへ差し替えます。34番で作ったclipを実際にGoblinへ使わせる段階です。

ステップ1
--------

Modsの中に以下のフォルダー階層を作る。

~~~text
Mods\Mod_YourName\Animations\Goblin Idle Override\Animation Clip Overrides
~~~

ステップ2
--------

mod.template_actor_targeted.jsonをコピーして、以下へ貼り付け、mod.jsonへ名前を変える。

~~~text
Mods\Mod_YourName\Animations\Goblin Idle Override
~~~

ステップ3
--------

mod.jsonを以下にする。

~~~json
{
  "experimentalApiVersion": "v1",
  "type": "animationClipOverride",
  "target": "Goblin_AC Edit",
  "variant": "",
  "entry": "Animation Clip Overrides"
}
~~~

targetはアニメーションセットIDです。patroller.goblinではありません。

他のactorのtargetを探す方法：

1. StreamingAssets\AnimationController_Detailを開く。
2. 目的のactorフォルダーを探す。
3. _controller.jsonを抜いたファイル名がtarget。
4. Goblin_AC Edit_controller.jsonならtargetはGoblin_AC Edit。

ステップ4
--------

Animation Clip Overridesフォルダーの中にmap.jsonを作る。

~~~json
{
  "priority": 100,
  "overrides": {
    "items": [
      {
        "clipSlot": "Goblin_Idling.main",
        "clipId": "mod.yourname.goblin_idle"
      }
    ]
  }
}
~~~

ステップ5
--------

map.jsonを理解する。

1. "priority": 同じslotを複数Modが差し替える場合、大きい方が勝つ。
2. "clipSlot": 差し替えるcontroller slot。
3. "clipId": 代わりに再生するclipId。

clipSlotは以下の形です。

~~~text
stateId.slotName
~~~

Goblin_Idling.mainは、stateIdがGoblin_Idling、slotNameがmainという意味です。

ステップ6
--------

Goblin_Idling.mainの出所を確認する。

以下を開く。

~~~text
StreamingAssets\AnimationController_Detail\Goblin\Goblin_AC Edit_controller.json
~~~

stateMotionBindingsの中に以下のような形があります。

~~~json
{
  "stateId": "Goblin_Idling",
  "clips": {
    "main": "goblin.goblin_idling"
  },
  "tuning": {},
  "motionParameters": {}
}
~~~

これがGoblin_Idling.mainというclipSlotを作っています。

ステップ7
--------

clipIdが存在することを確認する。

map.jsonに書いたclipIdは、どこかのanimation modまたはbuilt-inで読み込まれている必要があります。34番で作ったclipIdを使うのが一番簡単です。

自己完結型にしたい場合、map.jsonと同じフォルダーに_animations.jsonとPNGフォルダーも入れます。

~~~text
Mods\Mod_YourName\Animations\Goblin Idle Override
  mod.json
  Animation Clip Overrides
    map.json
    my_goblin_idle_animations.json
    mod.yourname.goblin_idle
      My_Goblin_Idle_0.png
      My_Goblin_Idle_1.png
~~~

トラブルシュート
--------

1. Mod Reportでは読み込まれるが見た目が変わらない：target違い、clipSlot違い、または他Modのpriority負け。
2. map.jsonが無いと言われる：entryフォルダー直下にmap.jsonがない。
3. animation source exists but cannot be built：clipIdはあるがPNG/spriteName/trackIdが壊れている。
4. Goblinが消える：差し替え先clipが描画できていない。

よくあるミス
--------

1. clipSlotにgoblin.goblin_idlingを書く。正しくはGoblin_Idling.main。
2. targetにpatroller.goblinを書く。正しくはGoblin_AC Edit。
3. map.jsonに存在しないclipIdを書く。
4. priorityを低くして他Modに負ける。
