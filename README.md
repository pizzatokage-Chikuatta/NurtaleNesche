# Nurtale Nesche Modding v1 Public Package

This package is for Nurtale Nesche experimental v1 modding.

v1 is experimental. JSON/PNG/audio/animation mods and Code/DLL mods are not guaranteed stable yet.

Start Here
--------

English:

~~~text
_Mod_Tutorial/EN/START_HERE.md
~~~

Japanese:

~~~text
_Mod_Tutorial/JP/START_HERE.md
~~~

Tutorial Source Of Truth
--------

The only tutorial source in this package is:

~~~text
_Mod_Tutorial
~~~

Do not use older docs/samples folders from previous packages.

What Is Included
--------

1. _Mod_Tutorial: EN/JP tutorials and reference data.
2. mod.json templates: manifest templates.
3. Developer Reference Packs: advanced reference material only.
4. dlls: reference DLLs for Code/DLL mod builds.
5. sdk_reference: public SDK/reference source.
6. tools/validator: validator source/reference.

Code/DLL Mods
--------

Read Tutorial Group 5 before writing Code/DLL mods.

The buildable reference project is:

~~~text
Developer Reference Packs/code_mod_reference_project
~~~

Unity Editor / AssetBundle Prefab Mods
--------

Read Tutorial Group 6.

The sample prefab package is:

~~~text
Developer Reference Packs/assetbundle_prefab_reference_pack/NurtaleNesche_AssetBundlePrefabSamples_v1.unitypackage
~~~

Validation
--------

Normal players should use the in-game validator from the main menu.

The validator source/reference is included under:

~~~text
tools/validator
~~~
